If there is an error in the values of the json document, the server will return a message with the path of the variable.

HTTP/1.0 400 BAD REQUEST
Content-Length: 1057
Content-Type: application/json
Date: Fri, 20 Mar 2015 14:41:01 GMT
Server: Werkzeug/0.10.1 Python/2.7.8+

{
    "agreement_date/day": {
        "message": "day is out of range for month",
        "value": 29
    }
}

If there is a extra key in the json document, the server will return this kind of message.

HTTP/1.0 400 BAD REQUEST
Content-Length: 1057
Content-Type: application/json
Date: Fri, 20 Mar 2015 14:41:01 GMT
Server: Werkzeug/0.10.1 Python/2.7.8+

{
    "name": {
        "message": "extra keys not allowed",
        "value": "value"
    }
}
